# FSMD - Fucking Sexy Music Downloader

- Add a folder named downloadedSongs in the root directory of scripts.
- Now run FSMD.py

  `python FSMD.py`

- Extra python modules and dependencies
  - mechanize
  - eyed3
  - pathlib
  - youtube_dl
  
- Install pip

  `sudo apt-get install python-pip`
  
- Install the modules

  `sudo -H pip install <module name>`
